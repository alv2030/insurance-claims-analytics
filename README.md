# Insurance Claims Analytics

## Overview

This project analyzes automobile insurance claim rates using demographic, geographic, and vehicle-related characteristics.

The objective is to evaluate claim frequency across policyholder segments and model insurance claim rates using statistical techniques commonly applied in insurance and actuarial analytics.

## Dataset

The dataset includes automobile insurance policy information such as:

- Number of policyholders
- Number of insurance claims
- Policyholder age group
- Vehicle category
- Residential district

## Objectives

- Analyze claim frequency across different policyholder groups
- Compare claim rates by age, district, and vehicle type
- Build statistical models for insurance claim rate analysis
- Apply exposure-adjusted modeling using policyholder counts

## Methodology

### Exploratory Data Analysis

- Calculated claim rates
- Compared claim frequency across categorical groups
- Visualized claim distributions by policyholder and vehicle characteristics

### Statistical Modeling

- Applied Generalized Linear Models
- Used Poisson Regression for claim frequency modeling
- Incorporated exposure adjustment using policyholder counts
- Interpreted model coefficients and claim rate relationships
